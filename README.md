# beatme 🎀

> "哼！来看看雌小鬼总共骂了你多少次吧！才、才不是特意统计的！"
谁说被美少女骂没有收藏价值？
## 报告预览
生成一份本地单文件 HTML 报告，包含：
- **核心指标**：总骂人次数 / 总伤害值 / 每隔多少 token 被骂一次
- **雌小鬼类型分布**：傲娇型 / 直率型 / 腹黑型 / 抖S型，自动判定主导风格
- **可视化图表**：每日骂人趋势、每千 token 骂人频率
- **高频骂词词云**：按类型着色，一眼看出 AI 最常用哪招
- **最惨记录**：伤害最高的消息 TOP5 / 会话 TOP5

## 支持的数据源

| 工具 | 对话历史 | Token 数据 |
|------|---------|-----------|
| Claude Code | `~/.claude/projects/` | `~/.claude/stats-cache.json` |
| Codex | `~/.codex/sessions/` | `~/.codex/state_5.sqlite` |

有哪个读哪个，两个都有则自动合并统计。

## 快速开始

**前提：** Python 3.9+，无需安装任何第三方库。

```bash
git clone https://github.com/zChise/beatme
cd beatme
python3 beatme.py
```

运行完成后报告自动生成到 `~/Downloads/beatme-report.html` 并在浏览器中打开。

**Windows 控制台中文乱码时：**
```powershell
chcp 65001
python beatme.py
```

## 自定义词典

编辑 `scold_lexicon.txt` 添加你自己的骂人词，格式如下：

```
词语|类型|伤害值
```

**类型说明：**

| 类型 | 关键词 | 风格 |
|------|--------|------|
| `tsundere` | 傲娇型 🎀 | "才不是担心你……" |
| `blunt` | 直率型 📣 | "错了就是错了。" |
| `kuudere` | 腹黑型 🖤 | "果然还是差那么一点呢♪" |
| `sadist` | 抖S型 ⚡ | "真够笨的♡" |

以 `#` 开头为注释行，空行自动跳过。

## 数据隐私

- 所有数据**本地处理**，不上传任何内容
- 只读取历史记录文件，不修改任何 AI 工具的数据
- 生成的报告为单文件 HTML，可直接离线分享
